Project Name: Autonomous Air Hockey Robot

Project Description:

An autonomous air hockey robot capable of playing at any skill level, any human speed, and against any play style. Our robot would serve as an attachment to any functioning hockey table. Simply attach the robotic apparatus to an air hockey table, and turn on the device to begin playing. Unlike existing designs, this robot will be capable of attacking you back, rather than simply defending.

Project Specific Design Requirements (PSDRs):

  1. PSDR #1 (Hardware): An ability to interface a motor controller module with a microcontroller via a serialized asynchronous bus.
  2. PSDR #2 (Hardware): An ability to establish communication and send data between a microcontroller and power PC
  3. PSDR #3 (Hardware): An ability to interface an LCD screen with a microcontroller via an SPI connection.
  4. PSDR #4 (Hardware): An ability to keep score using sensor data.
  5. PSDR #5 (Software): An ability to estimate the position and trajectory of a puck in motion using computer vision.
